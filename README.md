# 🌍 Country Info CLI Tool

A simple Python command-line tool that fetches real-time country information using a public REST API.

---

## 🚀 Features

- Fetch country data from an external API
- Display:
  - Country name
  - Capital
  - Region
  - Population
- Handles:
  - Invalid input
  - API errors
  - Network issues
- Clean and formatted CLI output

---

## 🛠️ Tech Stack

- Python
- requests (HTTP API calls)
- REST API (https://restcountries.com)
